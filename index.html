Ferramentas Utilizadas

Nesse módulo, vimos a instalação passo a passo do Visual Studio Code.

Aprendemos também como instalar extensões, além de que foram passadas algumas extenções úteis, como o Live Server.

Inspetor de Elementos

Nesse módulo, aprendemos a importância do uso do Inspetor de Elementos para estudo.

Ele nos mostra tudo que está na página, além de ser possível também fazer alterações no código. Essas alterações não são permanentes, e ao atualizar a página, as configurações são recarregadas.

Mas ela nos ajuda a ter uma ideia de como ficaria a página com essas alterações.

Estrutura Básica do HTML

Nesse módulo aprendemos sobre a estrutura básica do HTML. Ele é composto por três estruturas básicas:

  1. HTML
  2. Head
  3. Body
HTML

É tudo que está dentro da página, com todas suas configurações e estilos.

Head

São as informações de cabeçalho da página. Tudo que o navegador precisa ler primeiro antes de mostrar a parte visual do site ao cliente.

Body

É a parte visível do site ao cliente.

Essa parte que você está visualizando é o body.

Falando sobre Tags

Nesse tópico, nos foi ensinado sobre as mais diversas tags que o HTML possui, ensinando como temos uma imensidão de tags que podemos usar, de acordo com o que nosso projeto pede.

Atributos das Tags

Nos foi explicado que existem alguns atributos que são comuns a todas as Tags, como o atributo id por exemplo.

Também existem atributos que são específicos para algumas tags, como os atributos das tags img e a.

Textos

Há uma infinidade de tags para texto, além dos atributos que ajudam a estilizar elas dentro do próprio HTML.

Essas tags de texto devem ser escolhidas de acordo com o projeto que se está desenvolvendo.

Listas Ordenadas e Não Ordenadas

Listas Ordenadas

São as listas em que os itens ficam de forma ordenada, com o marcador número do lado.

Exemplo:
  1. Exemplo 1
  2. Exemplo 2
  3. Exemplo 3

Não Ordenadas

São listas em que os itens ficam com um marcador, sem o número.

Exemplo:
  • Exemplo 1
  • Exemplo 2
  • Exemplo 3
